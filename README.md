# Le Donjon de Hodor

Petit jeu web statique en HTML, CSS et JavaScript.

## Lancer en local

Ouvrir `index.html` dans un navigateur.

## Publier sur GitHub Pages

1. Mettre tout le contenu de ce dossier dans un depot GitHub.
2. Dans GitHub, ouvrir `Settings` > `Pages`.
3. Choisir `Deploy from a branch`.
4. Selectionner la branche `main` et le dossier `/root`.
5. Sauvegarder, puis attendre que GitHub donne l'adresse publique.

Le fichier `.nojekyll` est volontairement present pour que GitHub Pages publie les assets sans transformation.
